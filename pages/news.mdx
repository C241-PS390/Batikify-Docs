# Get News from Google

## Functions

This is used to get all news of Batik from Google

## How to Use

### Endpoint

`/encyclopedia?searchQuery={searchValue}&page={page}`

### Method

`GET`

### Request Body

1. `Headers`

   | Key           | Value                              |
   | ------------- | ---------------------------------- |
   | Authorization | loginTokenGeneratedFromLoginMethod |

2. `Query`

   | Key         | Value       |
   | ----------- | ----------- |
   | searchQuery | searchValue |
   | page        | page        |

### Error Response

1. General error for news

   ```json
   {
     "status": "error",
     "message": "errorMessage"
   }
   ```

   Status Code: `500`

2. User is not found

   ```json
   {
     "status": "fail",
     "message": "Gagal autentikasi token",
     "error": "Pengguna tidak ditemukan"
   }
   ```

   Status Code: `404`

3. Token is expired

   ```json
   {
     "status": "fail",
     "message": "Gagal autentikasi token",
     "error": "Token kadaluarsa"
   }
   ```

   Status Code: `401`

4. There are no token provided

   ```json
   {
     "status": "fail",
     "message": "Gagal autentikasi token",
     "error": "Tidak ada token"
   }
   ```

   Status Code: `403`

5. Invalid token

   ```json
   {
     "status": "fail",
     "message": "Gagal autentikasi token",
     "error": "Token invalid"
   }
   ```

   Status Code: `400`

6. General error for authentication

   ```json
   {
     "status": "fail",
     "message": "Gagal autentikasi token",
     "error": "generalError"
   }
   ```

   Status Code: `500`
