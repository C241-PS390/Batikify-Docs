## What's This?

Welcome to our API Documentation page! This API is designed to support the development of a mobile application that can classify various types of batik through the camera or user-uploaded images. :)

## Features

Our API offers the following key features:

1. Image Classification: Upload images or use the camera to classify and identify different batik patterns.
2. Detailed Information Retrieval: Access comprehensive information about each batik type, including its name, origin, history, and philosophy.
3. User-Friendly Integration: Easy to integrate with your mobile application, providing seamless functionality and user experience.
